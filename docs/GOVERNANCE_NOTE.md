# Governance Note v0.1

ATTICUS MPPS v0.1 is an environmental observation prototype.

## Rules

1. Environmental observation only.
2. No person detection.
3. No vehicle detection.
4. No face, license plate, identity, or tracking analytics.
5. No autonomous external action.
6. All outputs are candidate observations.
7. Raw inputs are locally stored.
8. Every derived packet must link to raw provenance.
9. Human review is required for anomaly candidates.
10. Outdoor deployment requires separate governance review.

## Deployment Boundary

Desktop v0.1 is not approved for outdoor continuous operation. Outdoor operation requires a separate review covering field of view, horizon masking, retention, access control, incident handling, and operator override.
